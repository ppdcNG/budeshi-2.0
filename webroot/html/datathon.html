BUDESHI DATATHON
Budeshi Datathon is intended to be a fun, collaborative and rewarding event comprising of a number of activities/challenges centered around the use of our civic tech tool Budeshi. These activities will be inclined towards procurement monitoring, data analysis, better value for money and media advocacy.
Winners will be announced after collation and evaluation of submissions on the 7th September 2018

Budeshi Chart Challenge
This challenge is intended to engage participants in the use of budeshi as a tool for analysis. Its involves participants to create 7 different visualisations using data from Budeshi strictly. This can be done by mixing different parameters on the project finder and charts tab which are features on Budeshi.
  1. Go to www.budeshi.ng
  2. Click on the charts tabs
  3. Use the projects finder to select parameters for the charts. E.g, location, MDA, e.tc. Note you can select one or more parameter
  4. Click on “Find Project”
  5. Click on the right of the Chart and select the file format to export.
  6. Repeat step 3 - 5 to get different type of charts.
  7. Submit your Challenge to www.budeshi.ng/Datathon/
The first participants that submit the correct charts for all of the cases will be the winners of #50,000 Cash prizes. Submit here

Writers Challenge
  1. Go to www.budeshi.ng
  2. Click on the charts tabs
  3. Explore procurement data using the features on Budeshi ( project search tools and charts )
  4. Come up with a story or blog post related to procurement data sourced from budeshi
  5. Submit your Challenge to www.budeshi.ng/Datathon/
Criteria for selection
  1. Articulation (How well the story was pieced together)
  2. Relevance and relativity (How much does it relate to the present issues in governance and procurement)
  3. Reference to Budeshi (How well does the story put Budeshi on the spotlight? )
